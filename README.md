# cs81-module3a-github
cs81, sect. 1233 - assignment 3A: Git &amp; GitHub

An assignment the help teach how to use Git and GitHub.
This repository is (I believe) just a "first test" to familiarize myself with all the functions of the two programs.

--- FUNCTIONS ---

greet(name): Takes in a string and returns a string that is a simple greeting using the provided input.

farewell(name): Takes in a string and returns a string that incorperates the inputted string as a goodbye message.

canDrink(age): Takes in a number and determines if a person of that age could legally drink.

appease(favorite): Takes in a string and returns a string stating telling the user that it's our favorite thing too.

ourAgeDiff(age): Takes a number and, based on my current age, determines if the user would be my junior, senior, or if we're the same age.
